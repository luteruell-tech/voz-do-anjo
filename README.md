# 🎙️ Voz do Anjo

> Mensagens que tocam a alma

Aplicativo web progressivo (PWA) que entrega mensagens diárias de fé, força e amor, no dia e horário escolhidos pelo usuário.

## ✨ Funcionalidades

- 📱 **PWA instalável** - Funciona como app nativo no celular
- 🎧 **Player de áudio** - Mensagens em áudio direto no app
- 🔔 **Agenda personalizada** - Configure dia e horário das mensagens
- 🎨 **Design elegante** - Tipografia clássica com animações suaves
- 🔐 **Painel admin** - Gerenciamento de áudios com upload e gravação direta

## 🗂️ Estrutura

```
voz-do-anjo/
├── index.html        # App principal (5 telas com animações)
├── admin.html        # Painel admin (senha: mateus1905)
├── manifest.json     # Configuração PWA
├── sw.js             # Service Worker (cache offline)
├── icon-192.png      # Ícone 192x192
└── icon-512.png      # Ícone 512x512
```
